# Sales & Performance Analysis Dashboard (Power BI)

## Overview
This project focuses on analyzing sales and performance data using an interactive Power BI dashboard. The goal is to provide clear business insights related to revenue, profit, orders, and return rates through visual analytics.

## Objectives
- Analyze overall sales performance using key business metrics
- Identify trends, top-performing products, and regional performance
- Support business decision-making through data-driven insights

## Tools & Technologies
- Power BI
- DAX (Basic)
- Data Modeling
- Data Visualization

## Key Features
- Multi-page interactive dashboard
- Executive-level KPIs (Revenue, Profit, Orders, Return Rate)
- Dynamic slicers for product, region, and time-based analysis
- Drill-through views for detailed insights

## Dataset
The project uses a structured sales dataset similar to AdventureWorks for analytical purposes.

## Outcome
The dashboard helps in understanding sales patterns, identifying high-performing products, and tracking business performance over time.
