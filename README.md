# Sales & Performance Analysis Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard built to analyze sales performance using key business metrics such as revenue, profit, orders, and return rates.

The dashboard is designed to help understand trends, compare product and regional performance, and support data-driven business decisions.

---

## Key Insights
- Overall sales, profit, and order trends over time
- Top-performing products and categories
- Regional performance comparison
- Return rate analysis

---

## Tools Used
- Power BI Desktop
- DAX (Basic)
- Data Modeling
- Data Visualization
